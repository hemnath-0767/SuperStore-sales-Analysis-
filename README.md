# 📊 Superstore Sales Dashboard | Microsoft Excel Portfolio Project

> My first data analytics project built using Microsoft Excel to analyze Superstore sales performance through interactive dashboards, Pivot Tables, charts, and slicers.

---

## 📖 Overview

This project is an interactive sales dashboard created in Microsoft Excel using the popular Superstore dataset. It was developed as my first hands-on data analytics project to practice data cleaning, Pivot Tables, dashboard creation, and business reporting.

The dashboard allows users to explore sales, profit, product categories, and regional performance using interactive filters.

---

## 🎯 Objectives

- Analyze overall sales performance
- Track profit across different regions
- Identify top-performing states
- Compare product category sales
- Practice Excel dashboard development
- Build business reporting skills

---

## 📊 Dashboard Features

### Executive KPIs
- Total Revenue
- Total Profit
- Average Profit Margin
- Top Performing Region
- Best Selling State

- Year
- Product Category
- Region

### Visualizations
- Monthly Sales Trend
- Sales by Category
- Profit by State (Map)
- Top States by Sales
- Sales vs Profit Analysis
- Sub-category Profit Comparison

- Technology category delivered strong sales performance.
- Monthly sales trends highlighted seasonal variations.
- Profitability differed significantly across product sub-categories.
